Ls (List Contents)

    mkdir Asia

mkdir (multiple directory)

    mkdir Europe Africa America

ls (List contents)

    ls
    Asia Europe Africa America

cd (change directory)

    cd Asia
    


    pwd
    /home/michael/Asia
